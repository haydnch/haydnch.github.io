题目描述

给定一个字符串s和一个模式p，判断s能否被p所匹配。

‘.’ 匹配任意单个字符

‘*’ 匹配零个或多个前面的一个字符

动态规划

定义状态

  1. 定义二维数组dp[sLen+1][pLen+1]存储计算过程。dp[i][j]表示s中前i个字符是否能与p中前j个字符匹配。其中，s的前i个字符指s[0]到s[i-1]。显然 i 的范围从1到sLen，j 的范围从1到pLen。
  2. 初始化数组 dp（默认全为false）：
    • p和s都为空，即dp[0][0] = true
    • p为空，s不为空，已经默认为false
    • p不为空，s为空，要对dp[0][j]进行初始化（p满足一定条件时，dp[0][j]=true）
    • p和s都不为空，就进入动态规划过程
  3. 对 s 和 p 都从1开始遍历。p 中第 j 个字符只能是p[j-1] = {a-z, '.', '*'} 三种情况。

状态转移方程

  • 如果p[j-1]是正常字符(a-z)并且与s[i-1]相等，或者p[j-1]='.'，那么两者是匹配的，因此只需要看两者前面的字符是否能匹配。所以，dp[i][j] = dp[i-1][j-1]。

  • 如果p[j-1]='*'，就要根据前一位字符p[j-2]来判断，因为*号使前面一位字符可以出现任意次。

    • p[j-2]!='.' 并且 s[i-1]!=p[j-2]，这时*让前面的字符出现 0 次，即抵消p[j-2]。因为跳过了p[j-1]和p[j-2]两位，所以，dp[i][j] = dp[i][j-2]。

    • 否则，如果p[j-2] == s[i-1]或者p[j-2]=='.'，就要分三种情况讨论，即*让p[j-2]出现了0次、1次或者多次。
